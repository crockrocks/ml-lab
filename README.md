# ml-lab

This repository contains my ML_LAB codes and is divided into 3 parts ;  
1. Classification problems contains most of the work it includes things such as classification models such as `RandomForestClassifier` , `NaiveBayesClassifier` , `DecisionTree` , `PCA` , `KNearestNeighbour` , `Evaluation Metrics` . 
2. Clustering problems contains clustering model `KMeans`  , `Elbow Curve` . 
3. Regression problems contains regression models `LinearRegression` .
